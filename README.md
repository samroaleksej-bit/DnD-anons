# Карточки Судьбы — Планарные Энкаунтеры

Пять планарных энкаунтеров для D&D 5e, оформленных как карточки с иллюстрациями.

**Живой сайт:** `https://<твой-ник>.github.io/<название-репо>/`

## Энкаунтеры

| Карточка | План | Тип |
|---|---|---|
| «Живое Пламя» | Огня | Временный |
| «Сердце Горы» | Земли | Временный |
| «Голос Шторма» | Воздуха | Постоянный |
| «Прилив Забвения» | Воды | Временный |
| «Раскол Теней» | Тьмы | Эпический |

## Публикация на GitHub Pages

1. Создай новый репозиторий на GitHub
2. Загрузи все файлы в корень репо (сохраняя структуру папок)
3. Перейди в **Settings → Pages**
4. В разделе **Source** выбери ветку `main`, папку `/ (root)`
5. Нажми **Save**
6. Через ~1 минуту сайт будет доступен по адресу `https://<ник>.github.io/<репо>/`

### Через Git (терминал)

```bash
git init
git add .
git commit -m "Initial commit — планарные энкаунтеры"
git branch -M main
git remote add origin https://github.com/<ник>/<репо>.git
git push -u origin main
```

Затем включить GitHub Pages в настройках репозитория.
